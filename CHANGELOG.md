## [v0.1.0] - 2026-06-29 ([`d322f63`](https://github.com/plexusone/omniimage/commit/d322f63))

### Highlights

- Unified Go library for AI image generation with OpenAI and Fal AI providers
- Comprehensive MkDocs documentation site with API reference and guides

### Added

- Provider interface with `Generate`, `Edit`, `Variations`, and `Upscale` operations ([`e6696e6`](https://github.com/plexusone/omniimage/commit/e6696e6))
- Request/response types: `GenerateRequest`, `EditRequest`, `VariationsRequest`, `UpscaleRequest` ([`e6696e6`](https://github.com/plexusone/omniimage/commit/e6696e6))
- Image size constants: 256x256, 512x512, 768x1024, 1024x768, 1024x1024, 1024x1792, 1792x1024 ([`e6696e6`](https://github.com/plexusone/omniimage/commit/e6696e6))
- Quality (`standard`, `hd`) and style (`vivid`, `natural`) enums ([`e6696e6`](https://github.com/plexusone/omniimage/commit/e6696e6))
- Structured error types: `ErrNoProviders`, `ErrUnknownProvider`, `ErrNotSupported`, `ErrRateLimited`, `ErrContentPolicy` ([`cf11c53`](https://github.com/plexusone/omniimage/commit/cf11c53))
- `APIError` type with status code, error code, message, and provider name ([`cf11c53`](https://github.com/plexusone/omniimage/commit/cf11c53))
- Model constants for OpenAI (GPT Image 2, GPT Image 1, DALL-E 3, DALL-E 2) and Fal AI (FLUX Pro/Dev/Schnell, SDXL) ([`ed74f83`](https://github.com/plexusone/omniimage/commit/ed74f83))
- `GetModelInfo()` function returning model metadata (provider, max images, HD support, edit support) ([`ed74f83`](https://github.com/plexusone/omniimage/commit/ed74f83))
- OpenAI provider with image generation, editing (DALL-E 2), and variations (DALL-E 2) ([`7ba0d86`](https://github.com/plexusone/omniimage/commit/7ba0d86))
- Fal AI provider with image generation and upscaling via Clarity Upscaler ([`d20ca0d`](https://github.com/plexusone/omniimage/commit/d20ca0d))
- Negative prompts, seeds, inference steps, and guidance scale for Fal AI ([`d20ca0d`](https://github.com/plexusone/omniimage/commit/d20ca0d))
- Custom image dimensions via `ProviderOptions` for Fal AI ([`d20ca0d`](https://github.com/plexusone/omniimage/commit/d20ca0d))
- `Client` with automatic provider instantiation and capability checks ([`115e6ee`](https://github.com/plexusone/omniimage/commit/115e6ee))
- `NewProvider()` factory function with environment variable fallback for API keys ([`115e6ee`](https://github.com/plexusone/omniimage/commit/115e6ee))
- `GetCapabilities()` function for static provider capability lookup ([`115e6ee`](https://github.com/plexusone/omniimage/commit/115e6ee))
